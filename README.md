# Super AI Review - Complete SaaS Platform

## System Overview
Full end-to-end SaaS platform for automated AI review generation.

## File Structure

### Public Pages
- **home.html** - Landing page with features, pricing, CTA
- **admin-login.html** - Admin login page

### Onboarding
- **signup-wizard.html** - 5-step wizard:
  1. Account creation
  2. Business info
  3. Services (add more)
  4. Staff (add more)
  5. Payment & subscription

### Admin Dashboard
- **dashboard.html** - Main dashboard with stats
- **my-clients.html** - All clients with filters
- **payments.html** - Payment tracking & renewals
- **edit-client.html** - Edit client details

### Customer Pages
- **client-page.html** - Public review form
- **review-generated.html** - ChatGPT prompt display

## Features
✅ Public homepage with pricing
✅ Multi-step onboarding wizard
✅ Subscription plans (Monthly/Annual/3-Year)
✅ Payment integration
✅ Admin dashboard with revenue tracking
✅ Client management with expiry tracking
✅ "Add more" functionality for services/staff
✅ QR code system
✅ ChatGPT prompt generation

## Navigation Flow
1. home.html → signup-wizard.html → dashboard.html
2. Dashboard → my-clients.html / payments.html
3. QR scan → client-page.html → review-generated.html

## Start Here
Open **home.html** to begin!
